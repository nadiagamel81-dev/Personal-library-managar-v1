# Personal-library-managar-v1

it's an app you use to manage your books 

the features are :
1) add books with ratings from 10 
2) delete books
3) research for a book
4) edit rate of a book
5) research for a book by its rate
7) view the books with ratings
8) exit


important note : 
the app don't save the books ,if you need this feature you must wait for v2
